    Zhen CRM by Telerim

    A simple CRM system built on CakePHP 2.x. While this does not have the comlpicated features of enterprise CRMs, it has the basic features that most small businesses could use - such as contacts, tasks, "deals" and a calendar.

    Installation instructions are on GitHub.

    Requirements
    • PHP 5.2.6 or higher (required by CakePHP 2.x)
    • MySQL
    • mod_rewrite or equivalent turned on
    Login screen: Contacts index: Add a contact: Calendar: Add an event: Add a deal:
